# 🚀 Proyecto Semana 5 - Node.js + Express

Este proyecto corresponde a la **Semana 5** del curso, donde se trabaja la creación de un servidor básico utilizando **Node.js y Express**, implementando rutas GET y POST.

---

## 📌 Objetivo del proyecto

Aprender a:

- Crear un servidor con Node.js
- Usar Express para manejar rutas
- Manejar parámetros en URLs
- Recibir datos mediante POST
- Probar endpoints básicos

---

## ⚙️ Tecnologías utilizadas

- Node.js
- Express
- npm
- JavaScript
