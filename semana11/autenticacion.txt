El login es el proceso que permite identificar a un usuario antes de ingresar a una aplicación. Gracias al login, solo las personas autorizadas pueden acceder a ciertas funciones. Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar cambios importantes. Esto ayuda a mantener la seguridad del sistema. No se deben mostrar públicamente contraseñas, documentos de identidad, direcciones, teléfonos, tokens, claves de API ni otros datos sensibles de los usuarios.
