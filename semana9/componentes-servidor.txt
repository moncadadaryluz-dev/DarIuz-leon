Sistema operativo:
Es el software base que permite ejecutar Node.js y administrar los recursos del sistema.

Node.js:
Es el entorno que permite ejecutar JavaScript en el servidor.

Aplicación server.js:
Es el archivo principal donde se define la lógica del servidor y las rutas.

Puerto:
Es el canal por donde el servidor escucha las solicitudes, en este caso el 3000.

Logs:
Son registros que muestran información del funcionamiento del servidor, errores o eventos importantes.
